─── WHAT CHANGED IN 2.0 ───────────────────────────────────────

DESIGN SYSTEM
• Pure #080808 black canvas. No dark navy, no gradients anywhere on UI.
• Single accent colour: #F0EDE8 bone white. Everything else is muted grey.
• Playfair Display 900 for all h1/h2/h3 display text — cold, editorial, unexpected.
• Zero border-radius. Every card, button, modal, input is a hard rectangle.
• Navigation: flat black bar with border-top (not floating glass pill).
• All labels 9px / 0.28em letter-spacing / uppercase.

TRAIN SCREEN
• Muscle groups: vertical list with coloured left accent bars (GROUP_GLOW colours).
  Chest=orange, Back=blue, Shoulders=gold, Arms=red, Legs=green, Core=purple.
• Quick start section below (Mega Session, Cardio) at lower visual weight.
• Preset sessions (Push/Pull/Legs etc) as rows with muscle descriptions + count badge.
• Exercise picker: scrollable filter tabs at top, rows below (NAME + MUSCLE + ?/☆/+).
• No emoji on main UI elements — only muscle group icons retain colour.

LIVE SESSION
• Sticky slim header: exercise name label + sets progress (X/Y) + live session clock.
• Session clock starts at 0:00 when session starts, ticks every second.
• Done sets: one compact row "1 · 80KG × 8 ✓" — tap to undo.
• Active set: clear NOW badge, full-width LOG SET button (bone white).
• RIR collapsed by default — one tap to expand EFFORT section.
• Mid-session PR alert: toast + multi-buzz haptic when you beat a previous best.
• PR badge appears on the done set row inline.
• Superset: "UNLINK SUPERSET" / "+ SUPERSET WITH NEXT" buttons (no emoji).
• Superset connector: compact row between exercises (no text overflow).

SESSION SUMMARY (finish screen)
• 4-stat grid: TOTAL LOAD / MINUTES / SETS LOGGED / NEW PRs
• PR breakdown listed by name (with previous best shown).
• Streak + total sessions.
• Full SET LOG — every exercise, every set (weight × reps + RIR if logged).
• REPEAT THIS SESSION button rebuilds the session for next time.
• SHARE CARD button generates a redesigned share image.

SHARE CARD (makeFlexCard)
• Pure black canvas, bone white text, Playfair Display.
• Top + bottom bone white accent lines.
• 4-cell stat grid: TOTAL LOAD / SETS LOGGED / TOP LIFT / DURATION.
• PR list if any (max 3 shown).
• Streak ribbon at bottom.
• "EVOLVE · PRIVATE. OFFLINE. YOURS." watermark.
• No orange/teal glows or gradients.

HOME SCREEN
• Completed sessions section is now date-navigable with ← → arrows.
• Shows today by default; tap ← to browse previous days.
• Each logged session has a VIEW button that opens the full set log modal.
• The set log modal (openWorkoutDetail) matches the finish summary exactly.
• REPEAT THIS SESSION from the history modal too.

─── KNOWN LIMITATIONS ──────────────────────────────────────────

• Playfair Display on the share card requires the font to have loaded.
  The draw() waits up to 800ms via Promise.race([fonts.ready, timeout]).
  On very slow connections the card may fall back to Georgia serif.

• The live clock (startSessionClock) uses setInterval(1000).
  It stops on close/finish via stopSessionClock().
  If the user backgrounds the app mid-session the timer pauses (expected).

• openWorkoutDetail pulls from DATA.workouts which is the stored history.
  Sets shown = non-warmup sets with kg>0 or reps>0.
